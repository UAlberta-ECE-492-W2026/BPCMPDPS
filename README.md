# BPCMPDPS
To design and prototype a predictive monitoring and alerting system that reduces electricity cost risk for a commercial office building by identifying periods where forecasted power consumption coincides with high electricity prices.


How to run:
1. Download docker desktop
2. Run Docker desktop
3. Fill in the .env file.
4. Go into the project folder where the .env file is located
5. Type: **docker compose up --build**
6. To shut down, Type: **docker compose down**
7. Go into the frontend directory.
8. Type: npm run dev
